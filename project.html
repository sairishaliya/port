My Projects

Library Management System

This Library Management System simplifies library operations by managing book inventory, tracking borrowed books, and maintaining user data efficiently. Key features include user registration, book search, checkout and return functionalities, and fine calculation. This project focuses on enhancing the user experience through a clear and intuitive interface.

Hostel Management System

Developed as part of a summer internship, this desktop application optimizes hostel administration by automating tasks such as student registration, room allocation, fee management, and complaints handling. The system leverages Apache NetBeans IDE and a MySQL database via phpMyAdmin, with features including merit-based room allocation, comprehensive inventory management, and an intuitive fee collection module.

Placement Office Automation
